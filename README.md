# [AFKnights](https://github.com/fabriciovo/AFKnights)
A multiplayer RPG Idle Game, build your team, figth against enemies.

## Features
* Idle Game
* Combat in turns
* Multiple Characters
* Save all your account information in real time (Teams, Characters, Levels, etc.)

## Built With
* Phaser
* JavaScript
* Firebase
